# Introducción

Nest (NestJS) es un framerwork para crear aplicaciones de sevidor Node.js eficientes y escalables. Utiliza JavaScript progresivo, está construido con TypeScript y lo soporta completamente (aunque también permite a los desarrolladores programar en JavaScript puro), FP (Programación Funcional) y FRP (Programación Funcional Reactiva)

Internamente, Nest utiliza frameworks robustos para servidores HTTP como Express (el predeterminado) y opcionalmente, puede configurarse para usar Fastify

Nest proprociona un nivel de abstracción superior al de estos frameworks comunes de Node.js (Express/Fastify), pero también expone sus API directamente al desarrollador. Esto les brinda la libertad de usar la gran cantidad de módulos de terceros disponibles para la plataforma subyacente.

## Intalación

Para empezar, puedes crear la estructura básica del proyecto con la CLI de Nest o clonar un proyecto inicial (ambos métodos darán el mismo resultado).

Para crear la estructura básica del proyecto con la CLI de Nest ejecuta los siguientes comandos. Esto creará un nuevo direcotrio de proyectos y lo llenará con los archivos principales de Nest y los módulos de soporte, creando así una estructura base converncional para tu proyecto. Se recomienda crea un nuevo proyecto con la CLI de Nest para los usuario principales.

```bash
npm i -g @nestjs/cli
nest new project-name
```

`Hint`: Para crear un nuevo proyecto TypeScript con un conjunto de caraterísticas más estricto, pase el indicador `--stric` al comando `nest new`
