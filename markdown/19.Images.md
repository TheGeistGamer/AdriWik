# Imágenes

Es cierto que resulta bastante difícil idea una sitaxis "natural" para insertar imágenes en un documento de texto plano.

Markdown utliza una sintaxis para imágenes que se asemeja a la de los enlaces, permitiendo dos estilos: en línea y referencia.

La sintaxis para imágenes en línea es la siguiente:

![Alt text](/markdown/assets/life-death.JPG)

![Alt text](/markdown/assets/life-death.JPG "Título opcional")

Es decir

- Un signo de exlación: !
- Seguido de corchetes que contienen el texto de latributo alt de la imagen
- Seguido de paréntesis que contienen la URL o ruta de la imagen, y un atributo de título opcional entre comillas simples o dobles.

La sintaxis de las imágenes de estilo de referencia tiene este aspecto:

![Alt text][id]

Donde "id" es el nombre de una referencia de imagen definida. Las referencias de imagen se definen utilizando una sitaxis idéntica a la de las referencias de enlace:

[id]: url/to/image  "Atributo de título opcional"

Al momento de escribir esto, Markdown no tiene una sitaxis para especificar las dimensiones de una imagen; si esto es importante, simplemente puede usar las etiquetas HTML "img" normales.
