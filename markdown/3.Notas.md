# Notas

En casi todos los sentidos, Markdown es la sintaxis ideal para tomar notas. Lamentablemente, Evernote, OneNote, dos de las aplicaciones de notas más populares, actualmente no son compatibles con Markdown. La buena noticia es que varias otras aplicaciones de notas son compatibles con Markdown

- Obdidian es una popular aplicación para tomar notas de Markdown repleta de funciones.
- Simplenote es una aplicación básica y gratuita para tomar notas disponibles para todas las plataformas.

- Notable es una aplicación para tomar notas que se ejecuta en una variedad de plataforma

- Bear es una aplicación similar a Evernote disponible para dispositivos Mac e IOS. No utiliza Markdown exclusivamente de forma predeterminada, pero puedes habilitar el modo de compatiblidad de Markdown

- Joplin es una aplicación para tomar notas que respeta tu privacidad.
Está disponible para todas las plataformas.
