# Énfasis

Markdown trata los asteriscos (*) y los guiones bajos (_) como indicadores de énfasis. El texto envulto con un* on un_ se envolverá con una etiqueta HTML "em"; los que contenga dos ** o dos __ se envolverán con una etiqueta HTML "strong". Por ejemplo, esta entrada:

Si aparece algo como esto: `MD036/no-emphasis-as-heading:`
Estás usando negritas o cursivas como si fueran títulos

*asterisco simple*

*guiones bajos simples*

**asteriscos dobles**

**guiones bajos dobles**

```html
<em>asteriscos simples</em>

<em>guiones bajos simples</em>

<strong>asteriscos dobles</strong>

<strong>guiones bajos dobles</strong>
```

Puedes usar el estilo que prefieras; la única restricción es que se debe usar el mismo carácter para abrir y cerrar un tervalo de énfasis.

El énfasis se puede usar en medio de una palabra

Pero si rodeas un * o un _ con espacios, se interpretará como un asterisco o un guion bajo literal.

Para insertar un asterisco o un guion bajo literal en una posición donde normalmente se usaría como delimitador de énfasis, puedes usar una barra invertida para escaparlo:

\*Este texto está rodeado de asteriscos literales\*
