# Escape automático de caracteres especiales

En HTMl, hay dos caracteres que requieren un tratamiento especial < y &. Los corchetes angulares izquierdos se utlizan para iniciar etiquetas; el signo & se utiliza para denotar entidades HTML. Si desea utilizarlos como caracteres literales, debe escaparlos como entidades, por ejemplo, &lt; y &amp;.

El sino & es particular resulta problemático para los desarrolladores web. Si desea escribir sobre «AT&T», debe escribir «AT&amp;T».
Incluso es necesario escapar el signo & dentro de las URL. Por lo tanto, si desea enlazar a:

http://images.google.com/images?num=30&q=larry+bird

debes codificar la URL de la siguiente manera:

http://images.google.com/images?num=30&amp;q=larry+bird

### Alerta `MD034:` Bare URL used

`markdownlint` quiere que uses la sintaxis de Markdown para enlaces.

Markdown te permite usar estos caracteres de forma natural, encargándose automáticamente del escape necesario. Si usas un & como parte de una entidad HTML, permance sin cambios; de lo contrario, se convertirá en &amp;
