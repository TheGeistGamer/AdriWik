# Código

Para indicar un fragmento de código, enciérrelo entre comillas invertidas (`). A diferencia de un bloque de código preformateado, un fragmento de código indica código dentor deun párrafo normal. Por ejemplo: 

Utilice la función `printf()`.

producirá:

```html
<p>Utilice la función <code>printf()</code>.</p>
```

Para incluri un carácter de acento grave literal dentro de un segmento de código, puede usar varios acentos graves como delimitadores de apertura y cierre:

``Aquí hay una tilde grave literal (`)`.

lo cual producirá esto:

```html
<p><code>Aquí hay una tilde grave literal (`).</code></p>
```

Los delimitadores de acento grave que rodean un segmento de código pueden incluir espacios: uno después de la apertura y otro antes del cierre. Esto puermite colocar caracteres de acento grave literales al principio o al final de un segmento de código: 

Una sola tilde grave en un segmento de código: `` ` ``

Una cadena delimitada por acentos graves en un segmento de código: ```foo` `

Con un segmento de código, los signos de ampersand y los corchetes angulares se codifican automáticamente como entidades HTML, lo que facilita la inclusión de etiquetas HTML de ejemplo. Markdown convertirá esto:

Por favor, no utilice ninguna etiqueta `<blink>`.

```html
<p>Por favor, no utilice ninguna etiqueta<code>&lt;blink&gt;</code></p>
```

Puedes escribir esto:

`&#8212;` es el equivalente codificado en decimal de `&mdash;`.

producir:

```html
<p><code>&amp;#8212;</code>es el equivalente codificado en decimal de<code>&amp;mdash;</code>.</p>
```
