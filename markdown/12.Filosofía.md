# Filosofía

Markdown se concibió para ser lo más fácil de leer y escribir posibles.

Sin embargo, la legibilidad es primordial. Un documento con formato Markdown debe poder publicarse tal cual, como texto plano, sin que parezca que se le han añadido etiquetas o instrucciones de formato. Si bien la sintaxis de Markdown se ha visto influenciada por varios filtros de texto a HTML existentes - como Setext, atx, Textile, reStrucredText, Grutatext y EtText -, la principal fuente de inspiración para su sintaxis el el formato del correo electrónico en texto plano.

Por ello, la sintaxis de Markdown se compone exclusivamente de caracteres de puntuación, cuidadosamente seleccionados para que representen fielmente su significado. Por ejemplo, los asteriscos alrededor de una palabra se ven como *énfasis*. Las listas de Markdown se ven, como listas.
Incluso las citas textuales parecen fragmentos de texto citados, suponiendo que alguna vez hayas usado el correo electrónico.
