# Correo electrónico

Si envía muchos correos electrónicos y está cansado de los controles de formato disponibles en la mayoría de los sitios web de proveedores de correo electrónico, le alegrará saber que existe una manera sencilla de escribir mensajes de correo electrónico utilizando Markdown, Markdown Here es una extención de navegador gratuita y de código abierto que convierte texto con formato Markdown en HTML listo para enviar
