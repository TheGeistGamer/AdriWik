# Sabores de rebajas

Uno de los aspectos más confusos del uso de Markdown es que prácticamente todas las aplicaciones de Markdown implementan una versión ligeramente diferente de Markdown. Estas variantes de Markdown se conocen comúnmente como sabores. es su trabajo dominar culaquier versión de Markdown que haya implementado su aplicacón.

Para comprender el concepto de sabores de Markdown, puede ser útil pensar en ellos como dialectos de idioma. La gente de la ciudad de Nueva York habla inglés igual que la gente de Londres, pero existen diferencias sustanciales entre los dialectos utilzados en ambas ciudades. Lo mismo ocurre con las personas que utilizan aplicaciones de Markdown. Usar JotBird para escribir con Markdown es una experiencia muy diferente a usar Ulysses.

En la práctica, esto significa que nunca se sabe exactamente qué quiere decir una empresa cuando dice que admite "MarkDown". Están hablando sólo de los elementos de sintaxis básicos, o de todo los elementos de sintaxis básicos y extendidos combiandso, o de laguna combinación arbitraria de elementos de sintaxis? No lo sabrá hasta que lea la documentación o comience a utlizar la aplicación.

Si recién estás comenzado, el mejor consejo que puedo darte es que elijas una aplicación Markdown con buen soporte Markdown. Esto contribuirá en gran medida a mantener la portabilidad de sus archivos Markdown. Es posible que desee almacenar y tulizar sus archivos Markdown en otras aplicaciones y, para hacerlo, debe comenzar con una aplicación que brinde un porte. Puede utlizar el directorio de herramientas para encontrar una aplicación que brinde un buen soporte.
