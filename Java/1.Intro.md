# Java

`Java` is a popular programming language designed by `James Gosling` to be "written once, run anywhere" with a Java Virtual Machine (JVM).

Java is most known for powering Android OS and Android Development. It's also used for video games, websites, and the language of choice in high schools and college CS classes around the world.

Java files have the `.java` extension

## Start

```java
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}
```

To run any line of Java code, you need a `class` and a `method`. In the example below, we have a class called `HelloWorld` and a `main()` method

The message we want to display `Hello World` should be inside the parentheses and surrounded by double quotes, and the command should end with a semicolon, `;`. Otherwise, our program will not work!

## Comments

Comments, while being invisible to the program, are lifesavers! 🛟

As developers, sometimes we need to add clarity to code, add context to another programmer, or even temporarily remove code.

### Single-line Comments

In Java, a single-line comments has two `//` forward slashes and looks like this:

```java
// Welcome mesasge for users
System.out.println("Hey there, friend!");
```

### Multi-line Comments

We can also use comments across multiple lines of code:

```java
/*
  System.out.println("The location can’t be updated.");
  System.out.println("The location is not up to date.");
*/
```
