# User Input

Hava your noticed that all of the Java programs we've written so far are only based on what you write to the program? Our java programs get much more intersting when th users get ot charge it!

```java
  import java.util.Scanner;

  public class Greetings {
    public static void main(String[] args) {
      Scanner scanner = new Scanner(System.in);

      System.out.print("Enter yourt name: ");
      String name = scanner.nextLine();

      System.out.println(name);
    }
  }
```

## How it works

This `import` statement tells Java we want to use the scanner.

```java
  import java.util.Scanner;
```

The scanner **object**: `Scanner scanner = new Scanner(System.in)` sets up thescanner to read input from the user.

In our program example, we used the `scanner.nextLine()` method to read strings, but we can also use

- scanner.nextDouble()
- scanner.nextBoolean()
- scanner.nextInt()

to read and save double, boolean and Int data tuypes to a variable

## Random

Java can use the `Random` class to generate random number form the `java.util` package to try and guess your number!

We'll first use the `import` statement and then create an instance to use it in our program.

```java
import java.util.Random;

Random rand = new Random();

int guess = rand.nextInt(11);
System.out.println("The computer guesses: " + guess); // 0-10
```

`rand` can now be used in our program to generate new integers from 0 to any number minus 1. 

```java
int coin = rand.nextInt(2);
int dicey = rand.nextInt(6);
int lottery = rand.nextInt(60);
```

The `Random` class also has the same functionality for doubles and booleans:

- `nextDouble()`: returns a `double` form 0.0 to 1.0
- `nextBoolean()`: returns true of false
