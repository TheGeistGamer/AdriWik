Like Arrays, `ArrayLists` hold data in collections or groups to help mange information in our code.

The difference is, unlike Arrays, the are mutable, meaning the length of the `ArrayList` can change. This is helpful when you're working with data that is consistently changing.

To create on `ArrayList`, declare an `import` statement (similar to `until.Arrays` for Array methods).

```java
import java.until.ArrayList

public class HelloWorld {
	public static void main(String[] args) {
		// Create a new ArrayList
		ArrayList<String> cities = new ArrayList<>();
	}
}
```

## Adding, Modifying, and Removing

To add elements to an `ArrayList`, we manually add elements using `.add()` and access them `.get()`, using the `ArrayList` index

```java
cities.add("chicago");  // add "Chicago to index 0"
cities.add("New York City"); // add "New York City to index 1"

System.out.println(cities.get(0)); // access index 0: “Chicago”
System.out.println(cities.get(1)); // access index 1: “New York City”
```

To modify or update an element is an `ArrayList`, you'll use `.set()`, which take the index of the element you want to change, and the new element as arguments.

```java
cities.set(1, "Boston");
```

Removing an element works similarly, taking in the index you wantto remove as an argument with `.remove()`

```java
cities.remove(0);
```

You can clear the list altogether with `.clear()`

```java
cities.clear();
```
