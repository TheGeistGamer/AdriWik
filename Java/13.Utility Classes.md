
# Utility Classes

A **Utility class** is a class that provides a collection of static methods!

```java
import java.until.Scanner;
```

Because we've been making `scanner` objects, we know we have been creating instances of the `Scanner` class.
Just like the classes we've been creating ourselves, the `Scanner` is a class that is already built into Java!

Java has hundreds of utility classes built into its standard library (Calles the Java API) so that developers can use and access them with a `import` statement!

The **utility package** is package that holds several utility classes

## Math in Java

To use the math class, you do no need an import statement, since it is already a part of the `java.lang` package, which also includes `String` 

You'll use dot notation to write `Math` and the name of the method you're using, `.name()`

```java
public class Rockets {
	public static void main(String[] args) {
		double result = Math.sqrt(25)
		int maxNum = Math.max(8, 15);
		int minNum = Math.min(8, 15);
		
		System.out.println("Square root: " + result);
		System.out.println("Max ✈️: " + maxNum); // 15 
		System.out.println("Min 🚀: " + minNum); // 8
	}
}
```

## public, private, protected

**Access modifiers** control who can access what method, variable, or class.

- `public`: Methods/variables/classes accessible by any class
- `private`: Methods/variables/classes accessible only inside the dame class
- `protected`: Methods/variables/classes visible only to classes in the same package
- No modifier: Methods/variables only accessible in the same package

The generate standard for a simple Java program has public classes with private variables and public methods.

We first create a public class `Account` where we add `private` variables

```java
public class Account {
	// --- Private fields ---
	private String residentName; // Player's name
	private String isLandName;   // Island name
	private int bellsBalance;    // Stored in Bells - AC's currency
}
```

We the set `private` variables inside our public constructor so we can access them in our Main class.

```java
public Account(String residenName, String isLandName, int startingBells) {
	this.residentName = redienteName;
	this.islandName = islandName;
	this.bellsBalance = startingBells;

```

Then, we can create `public` methods to use in our `Main` class to use whenever we need to

```java
public void depositBells(int amount) {
	if (amount > 0) {
		bellsBalance += amount;
		System.out.println(residentName + " deposited " + amount + " Bells!");
	} else {
		System.out.println("Invalid deposit amount.");
	}
}
```

We need access modifiers to protect and organize our code.
Access modifiers help keep your code clean, safe, and flexible
