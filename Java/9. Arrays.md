## Indexing

A Java arrays holds any collection of data that you may need to create!

```java
String[] continents = {"Asia", "Africa", "North America", "South America", "Australia", "Europe", "Antarctica"}; // Total Length: 7
```

We know that the length of this array is 7. 
**The length of this array once initialized cannot be changed, but the elements themselves can!**

Now, let's say we need to access the string `"North America"`. It's the third element in our list
In comes position/index 0! In programming we begin counting at 0

```java
char[] vowels = ['a', 'e', 'i', 'o', 'u']
// Index: 0 1 2 3 4
```

Even though 'o' is the fourth element in the array, it has the index 3

System.out.println(continents[3]);
System.out.println(continents[4]);

## Updating

While we can't change the length of an array, we can update the elements inside of it by reassigning the value at a specified index

```java
String[] champions = {"Jinx", "Vi", "Caitlyn", "Ekko", "Jayce", "Viktor", "Heimerdinger"};

champion[0] = "Powder";

System.out.println(champion[0]); // Powder
```

To add new element to an empty array, we can assign a value to an index that doesn't exist yet

```java
int[] emptyArray = new int[5]; // Create an empty int array with 5 elements

emptyArray[0] = 100
emptyArray[1] = 200

System.out.println(emptyArray[0]); // 100 
System.out.println(emptyArray[1]); // 200
```


## Length

The `.length` **array property** can return the length of the array

```java
String[] a24Films = {"Heretic", "I saw the TV Glow", "Talk to Me", "Lady Bird", "Past Lives", "Midsommar", "Aftersun", "Green Room", "Uncut Gems", "Everything Everywhere All at Once"};

System.out.println(a24Films.length);
```

This would ooutput:

```java
10
```


By using the array's name followed by the `.length` property, we can easily access the number of element in the array and use that value in expressions or loops throughout the code.

We are using a `for` loop to iterate throungh the array and print each element:

```java
for(int i = 0; i < a24Films.length; i++) {
	System.out.println(a24Films[i]);
}
```

## java.until.arrays

Now that we can crate, update and use arrays in loops, we have even more to dicover about organizing data with arrays

The `java.until.Arrays` **utility class** is a useful toolbox that lets us transform arrays and makes our life a bit easier when trying to accomplish certain tasks, using tools called **static methods**. This means that instead of writing a bunch of code ourselves to do comoon taks, we can call there ready-made tools (methods) directly form the Arrays class.

here are som of the most common tasks that we may want to perform with code:

## .sort()

The `.sort()` method takes in an array and sorts its elments is ascending order form smallest to larges for numbers, or alphabetically for string

Sorting numbers:

```java
int[] numbers = {4, 2, 7, 1};
Arrays.sort(numbers); // [1, 2, 4, 7]
```

Sorting strings:

```java
String[] names = {"Cait", "Memo", "Andy"};
Arrays.sort(names); // ["Andy", "Cait", "Memo"]
```

## .equals()

The `.equals()` method allows you to check if two arrays contain the same elments, being equal, which returns a boolean either `true` of `false`

```java
int[] a = {1, 2, 3};
int[] b = {1, 2, 3};
int[] c = {3, 2, 1};

System.out.println(Arrays.equals(a, b)); // Output: true System.out.println(Arrays.equals(a, c)); // Output: false
```

As an note, these are only available with the `java.util.Arrays` import statement
