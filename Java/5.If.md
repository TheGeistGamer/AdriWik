# If statements

You used an `if` statement to print something only if a certain condition was met. If the statementis false, anything inside the `if` block doesn't happen.

This is the exat definition of an `if` statement

```java
  int score = 85;

  if (score > 65) {
    System.out.println("You passed the test!");
  }
```

```java
  int peppers = 1;
  boolean isSpicy = peppers > 3;

  if (isSpicy) {
    System.out.println("Careful! this is spicy fried rice.");
  }
```

The example above works in the same way. Only this time, we are saving the conditional statment `peppers > 3` in a boolean variable that holds either `true` or `false` value

## Else

There are times when you want to reachy a conclusion if the `if` statement is *false*. We can use the `else` keyword.

```java
  int spiceLevel = 1;
  boolean isSpicy = spiceLevel > 3;

  if (isSpicy) {
    System.out.println("Careful! This is spicy fried rice.")
  } else {
    System.out.println("Spice tolerance where?");
  }
```

The `else` block makes sure that the `if` statement will always come to a drecision, either one or the other.

### Example

```java
  public class HelloWorld {
    public static void main(String[] args) {
      String pet = "cat";
```
