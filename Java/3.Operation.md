# Operators

Java has the following operators to conduct mathematical operations with code:

- `+` Addition
- `-` Subtraction
- `*` Multiplication
- `/` Division
- `%` Modulo (return the remainder)

We can use these operators to chage vairables and conduct math problems! Here is an example of how we use them:

```java
  double result = 0;

  result = 8 + 2;   // result is now 10
  result = 8 - 2;   // result is now 6
  result = 8 * 2;   // result is now 16
  result = 8 / 2;   // result is now 4
  result = 8 % 2;   // resutl is now 0

  System.out.println(result); // 0
```

Example: 

```java
  double originalPrice = 60.0;
  double discount = 20.0;

  // Apply discount
  double finalPrice = originalPrice - (originalPrice * (discount / 100));

  System.out.println(finalPrice);
```

Just like in math, parentheses are the highest order of operations, and calcultations inside of them are performed first.
