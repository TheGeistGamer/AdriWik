# Relational Operators

We often compare two values inside conditionals! We've seen the `>` symbol, which is a type of operator called a **relational operator** that compares values!

Below are the other relational operators we use:

- `==` equal to
- `!=` not equal to
- `>` greater than
- `<` less than
- `>=` greater thatn or equal to
- `<=` less than or eaqul to

```java
  int onlinePrice = 11;
  int storePrice = 11;

  if (onlinePrice == storePrice) {
    System.out.println("The prices are the same!");
  }
```

## else if

We've been workin with programs that output either one conclusion with an `if` block or a different conclusion with an `else` block. You might not be surpreiced to discover that we can expand this logic tree to make even more complex decisions.

```java
  int temp = 90;

  if (temp < 40) {
    System.out.println("It may be snowing! ❄️")
  } else if (temp < 80) {
    System.out.println("Enjoy the weather! 👌")
  } else {
    System.out.println("Okey... It is getting a little hot in her. 🔥")
  }
```
