
Create a file

```bash
$ mkdir <file name>
```

Crear una carpeta con espacios

```bash
$ mkdir <nombre con espacios>

$ mkdir <name> \ <name>
```

Eliminar una carpeta solo sirve para eliminar repositorios vacio

```bash
$ rmdir <name>
```

Crear un archivo 

```bash
$ touch <name>

$ touch <name> \ <name>
```

Eliminar un archivo

```bash
$ rm <name>
```
