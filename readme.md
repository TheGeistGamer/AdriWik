# 📚 Programming Wiki

Repositorio personal que funciona como una wiki de programación, donde recopilo anotaciones, conceptos clave y ejemplos práctivos sobre distintos lenguajes y tecnologías.

## 🚀 Objetivos

Este proyecto nace como una base de conocimiento en constante crecimiento, con el objetivo de:

- Tener una referencia rápida y organizada
- Documentar aprendizajes del día a día
- Centralizar conceptos importantes
- Reforzar conocimientos mediante escritura

## 🧠 Contenido

Aquí encontrarás notas sobre:

- Lenguajes de programación
- Frameworks y librerías
- Buenas prácticas
- Patrones de diseño
- Snippets y ejemplos prácticas

## 🛠 Tecnologías cubiertas (ejemplo)

- JavaScript / Typescript
- NodeJS
- React
- NestJs
- Base de datos (SQL / NoSQL)
- Git y herramientas de desarrollo

> ⚠️ Nota: El contenido crece contantemente conforme aprendo nuevas tecnologías
