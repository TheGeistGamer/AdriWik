## Hello World

In Python, the `print()` function is used to tell a computer to 'talk'

The message we want to display should be inside the parentheses and surrounded by quotes. They can be double quotes `"` or sigle quotes `'`, but the opening and cloding quote marks have to be the same.

```python
  print('👋 Howdy')
```

In the example above, we instructed our program to print a message.

## Line by line

Python is run one line at a time, from top to bottom

We can output multiple messages by using multiple `print()` functions.
For. example, if we want to print out two simple greetings:

```py
  print('🌻 Morning')
  print('🙋🏽 Evening')
```

## Comments

Comments are very important in progrmaming because they are used to document what our code does. They are also used to disable part of the program.

When the program is run, the comments are ignored

In Python, we can create a comment using the `#` hashtag symbol:

```py
  # Printing out a message
  print('Hi')
```

On the first line, we created a comment. As a result, everything to the right of the hashtag `#` is ignored. The program continues to the next line

Python only ignores anything after the `#` on a line. So when a comment is placed towards the end of a line like this:

```py
  print('Hi')
```
