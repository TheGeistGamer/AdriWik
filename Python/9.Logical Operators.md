# Logical Operators

> **Logical operators**, also known as Boolean operatros, combine and evaluate two conditions. They are `and`, `or` and `not` operators:

- `and` returns `true` if both conditions are `True`, and returns `False` otherwise

- `or` returns `True` if at least one of the conditions is `True`, and `False` otherwise

- `not` returns `True` if the condition is `False`, and vice versa.

Some examples:

```py
  if hunger > 4 and anger > 1:
    print('Hangry')
```

If the `hunger` variable is greater than 4 *and* the `anger` variable is greater than 1, then the program prints "Hangry"

```py
  if coffee > 0 or bubble_tea > 0:
    print('😊')
```

If the `coffee` variable is greater than 0 or the `bubble_tea` variable is greater than 0, then the program prints a smiley face

```py
  if not tired:
    print('Time to code!')
```

If the `tired` variable is `False`, the program prints "Time to code"

- Else if they are tall enough, buy they don't have enough credits, print "You don't have enough credits"

- Else, print a message saying they have not met either requirement

```py
  height = int(input("What's your hight? "))
  credits = float(input("How many credits you have? "))

  if height >= 150 and credits >= 20:
    print('Enjoy the ride!')

    print('You are not tall enough to ride.')
  elif height >= 150 and credits < 20:
    print("You don't have enough credits")

```
