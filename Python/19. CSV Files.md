
# CSV Files

A CSV (comma-separated values) file is a common format for organizing data. Python provides a nifty `csv` module ro read and write data in this format.

Each row in a CSV file typically represents a record, and the values within a record are separated by commas. This simple structure makes CSV a widely used format for storing and exchanging data between applications.

## Reading CSV Files

Form the `csv` module, we use the `.reader()` method to create a CSV reader object that display our file content, row by row:

```python
import csv

# Open the CSV file in 'read' mode with UTF-8 encoding:
with open('example.csv', 'r', encoding='utf8') as file:
	csv_reader = csv.reader(file)
	
	for row in csv_reader:
		print(row)
```

Sometimes opening CSV files without specifying an encoding can cause decoding errors (especially in IDEs). If you're seeing strange errors when reading the file, try adding `encoding="utf8"` when opening the file. It helps Python properly read characters form the file.

The `reader()` method returns a list of rows where each row is a list of CSV data.

With a `for` loop, you can iterate through each row in the reader object.

## Writing to CSV Files

We use the `.writer()` method to create a CSV writer object, and then use its methods to write rows to the file

```python
import csv

data_to_write = [
	['Name', 'Age', 'Grade'],
	['Alice', 25, 'A'],
	['Bob', 22, 'B'],
	['Charlie', 28, 'A+']
]

# Open the CSV file in 'write' mode
with open('output.csv', 'w', newline='') as file:
	# Create a CSV writer object
	csv_writer = csv.writer(file)

	# Write the data to the CSV file
	csv_writer.writerows(data_to_write)
```

The writer object's `.writerows()` methods can parse something iteable and write it into the CSV file
