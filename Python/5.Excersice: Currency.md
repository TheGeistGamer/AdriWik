# Currency

We just hot home from a fun trip to South America, specifically Colombia, Peru and Brazil. There's some leftover cash in:

- 🇨🇴 Colombian pesos
- 🇵🇪 Peruvian soles
- 🇧🇷 Brazilian reais

Create a program that asks the user for the amount they have in pesos, soles, and reais and calculates the total in USD

The output should look somthig like this, but with different results:

```bash
  What do you have left in pesos? 5600
  What do you have left in soles? 105
  What do you have left in reais? 280
  413.0
```

```py
  pesos = float(input('What do you have left in pesos? '))
  soles = float(input('What do you have left in soles? '))
  reais = float(input('What do you have left in reais? '))

  colToUSD = pesos * 0.00027
  solesToUSD = soles * 0.29
  reaisToUSD = reais * 0.19

  total = colToUSD + solesToUSD + reaisToUSD

  print(total)
```
