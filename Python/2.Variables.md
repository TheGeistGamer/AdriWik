# Variables

In programming, `variables` are used fom storinig data values. Each variable has a name and holds a value.

The variable name can consist of letters, numbers, and the `_` underscore.

```py
  name = 'Jessica Jones'
  user_id = 234523389923
  progress = 0.34
  xp = 23
  verified = True
```

The `=` equal sign means assignment:

- We're assining the string value `Jessica Jones` to the variable `name`
- We're assining the number value `234523389923` to the variable `user_id`
- We're assining the number value `0.34` to the variable `progress`
- We're assining the number value `60` to the variable `xp`
- We're assining the true value `True` to the variable `verified`

We can also change the value of a variable or print it out:

```py
  xp = 70
  xp = 80

  print(xp) # Output: 80
```

Here, we are assigning the number value `70` to the variable `xp`. Then, we are reassingning the number value `80` to the same variable. And printing it out

## Data Types

### Intger

An integer, or `int`, is a whole number. It has no decimal point and contains the number 0, positive and negative counting numbers. If we were counting the number of people on the bus or the number of jellybeans in a jar, we would use an integer

```py
  year = 2023
  age = 32
```

### Float

A floating-point number, or a `float`, is a decimal number. It can be used to represent fractions or precise measurements if we were measuring the length and width of the couch, calculation the test scrore percentage, or storing a baseball player's batting average, we would use a `float` instead of an `int`

```py
  pi = 3.14159
  meal_cost = 12.99
```

### String

A string, or `str`, is used for storing text. String are wrapped in double quotes `"` or single quotes `'`

```py
  message = "good nite"
  username = '@snoopdog'
```

When we did printing in the first chapter, we printed a string data type

### Boolean

A Boolean date type, or `bool`, stores a value that can only be either true o false. In Python, it's capitalized `True` or `False`.

```py
  late_to_class = False
  cranky = True
```
